Functions

map (f, l[, ...])
Apply a function to all elements of a list, creating a new table of the results.

Parameters:

  • f function The mapping function.
  • l table The list.
  • ... Extra arguments to the mapping function f. (optional)

Returns:

    table The list of results.

Usage:

    list.map(function(x) return 2*x end, {1, 2, 3}) --> {2, 4, 6}
filter (f, l[, ...])
Apply a function to all elements of a list, if the function returns a truthy value the element is added to the new list.

Parameters:

  • f function The predicate function.
  • l table The list.
  • ... Extra arguments to the predicate function f. (optional)

Returns:

    table The filtered list

Usage:

    list.filter(
     function(x) return x % 2 == 0 end,
    {0, 1, 2, 3, 4, 5}
) --> {0, 2, 4}
zip (f, l1, l2)
Zips together two lists, calling the function argument on each pair of the list elements to produce the new value.

Parameters:

  • f function The zipping function.
  • l1 table The first list.
  • l2 table The second list.

Returns:

    table The resulting list.

Usage:

    l1, l2 = {1, 2, 3}, {5, 10, 15}
list.zip(
    function(a, b) return a * b end,
    l1, l2
) --> {5, 20, 45}
fold (f, a, l[, ...])
Performs a lefthanded reduction over a list, returning a value.

Parameters:

  • f function The reducer.
  • a The initial value of the accumulator.
  • l table The list.
  • ... Extra values to the reducer f. (optional)

Returns:

    The accumulated value.

Usage:

    function add(a, x) return a + x end
list.fold(add, 0, {1,2,3}) --> 6
find (f, l)
Finds the first value in the list which satisfies the predicate f.

Parameters:

  • f function The predicate.
  • l table The list.

Returns:

  1. The found value.
  2. The found key.

Or

    nil Nil if nothing satisfies the predicate.
each (f, l[, ...])
Calls a function on each element of a list in order.

Parameters:

  • f function
  • l table The list.
  • ... Extra arguments to f. (optional)
reverse (l)
Reverses a list returning a new list.

Parameters:

  • l table The list to reverse.

Returns:

    table A reversed copy of l.
shuffle (l)
Performs a Fisher-Yates out of place shuffle on l returning the shuffled copy.

Parameters:

  • l table The list.

Returns:

    table The shuffled copy of the list.
