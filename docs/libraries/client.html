Objects

client
The client state.

Fields:

  • options client_options The options used to instantiate the client. (copied)
  • api api.api The api state.
  • shards table The collection of shards the client is running.
  • loops table The collection of cqueues controllers associated with the client.
  • dispatch table The raw event dispatch handler table.
  • events table The events table, define your event handlers here.
  • cache table The cache table.

See also:

    client.create
client_options
Available options.

Fields:

  • token string The bot token.
  • sharding table A tuple {first, last} indicating the first and last shard id to be ran by this client.
  • compress bool A boolean to indicate whether the client should request payload compression from the gateway. (default false)
  • transport_compression bool A boolean to indicate whether the client shards should use transport compression. (default false)
  • large_threshold integer How many members are initially fetched per guild on start. (default 100)
  • auto_reconnect bool A boolean to indicate if the client should automatically connect if a reconnection is possible. (default false)
  • receive_timeout number The shard websocket receive timeout. (default 60)

See also:

    client.create
