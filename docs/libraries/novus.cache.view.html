Module novus.cache.view

Object for a read-only view into a table.

Functions

new (t, f, a) Construct a new view of the table t.
remove (view, value) Returns a view which can see all values inside the table, except the one specified.

Objects

view A view of a table.


Functions

new (t, f, a)
Construct a new view of the table t.

Parameters:

  • t table The table to view.
  • f function A function which is called with the key, value from t, its return value is the value view[key] returns, if it returns nil no value is returned.
  • a An argument to f.

Returns:

    view

Usage:

    v = view.new({1,2,3}, function(_, x) return 2*x end)
v[1] --> 2
remove (view, value)
Returns a view which can see all values inside the table, except the one specified.

Parameters:

  • view table The table (can be a view) to view.
  • value The value to ignore.

Returns:

    view

Usage:

    t = {1,2,3,4,5}
v = view.remove(t, 4)
t[4], v[4] --> 4, nil
remove_key (view, key)
Returns a view which can see all values inside the table, except the one accessed by the key specified.

Parameters:

  • view table The table (can be a view) to view.
  • key The key to ignore.

Returns:

    view

Usage:

    t = {"foo", "bar", "baz"}
v = view.remove_key(t, 1)
t[1], v[1] --> "foo", nil
__pairs (view)
View pairs iterator.

Parameters:

  • view view The view to iterate over.

Usage:

    v = view.remove({1,2,3}, 1)
for k,v in pairs(v) do print(k, v) end
 --> 2  2
 --> 3  3
flatten (view)
Flattens a view of views into a table.

Parameters:

  • view view The view to flatten.

Returns:

    table A plain table of the key value pairs the view can see.

Usage:

    v = view.remove({1,2,3}, 1)
view.flatten(v) --> {nil, 2, 3}
identity ()
An identity view function.
copy (t)
Creates a view copy of a table (or view). Shorthand for view.new(t, view.identity).

Parameters:

  • t table The table to copy.

Returns:

    view The view copy.

Objects

view
A view of a table. This is constructed by view.new. Its fields are the same as the table it’s viewing, except when the predicate map returns nil.
