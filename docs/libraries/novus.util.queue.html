filter (list, f[, ...])
Filters a queue in-place, from left to right. See novus.util.list.filter for filtering examples.

Parameters:

  • list queue The queue to filter.
  • f function The predicate function.
  • ... Optional arguments to f. (optional)

Returns:

    queue

Usage:

    q = queue.new():pushes_right(1, 2, 3, 4, 5, 6, 7, 8, 9)
q:filter(function(x) return x % 2 == 0 end)
for _, v in q:from_left() do print(v) end
--> 2
--> 4
--> 6
--> 8
