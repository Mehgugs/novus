Module novus.util.printf

Logging utilities.

See also:

    novus.util

Functions

info (str[, ...]) Logs to stdout, and the output file if set, using the INF info channel.
warn (str[, ...]) Logs to stdout, and the output file if set, using the WRN warning channel.
error (str[, ...]) Logs to stdout, and the output file if set, using the ERR error channel.
throw (str[, ...]) Logs an error using printf.error and then throws a lua error with the same message.
fatal (str[, ...]) Logs an error using printf.error and then exits with a non-zero exit code.
printf (str[, ...]) Logs to stdout, and the output file if set.
mode (m) Change color mode for writing to stdout.

Fields

fd An optional lua file object to write output to, must be opened in a write mode.

mode (m)
Change color mode for writing to stdout. You can use: – 3 for ¾ bit color – 8 for 8 bit color – 24 for 24 bit true color. Setting it to 0 disables coloured output.
