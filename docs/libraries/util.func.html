Functions

compose (a, b)
composes two functions. compose(f,g) = g(f(…)).

Parameters:

  • a function The inner function.
  • b function The outer function.

Returns:

    function The resulting function.
bind (f, a)
binds a single value into the first argument of given function.

Parameters:

  • f function The function to bind onto.
  • a The value to bind.

Returns:

    function The new function.
bindmany (f, ...)
Like bind but accepts many arguments, binding each onto the Nth argument of f.

Parameters:

  • f function The function to bind onto.
  • ... One or more values to bind.
call (f)
Creates a factory function from the given function.

Parameters:

  • f function The function to create a factory from.

Returns:

    function The factory function.

Usage:

    newprint = call(print)
myprint = newprint("Magicks says:")
myprint("Hello, world!") --> Magicks says:   Hello, world!
vmap (f, ...)
Performs a map over a vararg, returning a new vararg.

Parameters:

  • f function The mapping function.
  • ... The vararg of values.

Returns:

    The new vararg where each member i is f(i).
vfold (f, a, ...)
Performs a lefthanded reduction over a vararg, returning a value.

Parameters:

  • f function The reducer.
  • a The inital value of the accumulator.
  • ... The vararg of values.

Returns:

    The accumulated value.
vchop (first, ...)
Chops the last value off a vararg, and retuns it and the resulting vararg. You should call this function with a vararg vchop(…).

Parameters:

  • first The first vararg value.
  • ... The rest of the vararg.

Returns:

  1. The last value of the vararg.
  2. The vararg with the last value removed.
