Functions

init (options)
Creates a new api state for connecting to discord.

Parameters:

  • options table The options table. Must contain a token field with the api token to use.

Returns:

    api The api state object.
request (state, method, endpoint[, payload={}][, query={}][, files=nil])
Makes a request to discord.

Parameters:

  • state table The api state.
  • method string The HTTP method to use.
  • endpoint string The endpoint to connect to.
  • payload table An optional payload for PUT/PATCH/POST requests. (default {})
  • query table An optional table to convert into a url query string, appended to the URL. (default {})
  • files table An optional table of files to upload in a multi-part formdata request. (default nil)

Returns:

  1. boolean Whether the request succeeded.
  2. table The decoded JSON data returned by discord.
  3. An error if the request did not succeed.
route_path (state, ...)
Request a specific resource. Function name is the routepath in snake_case Please see the discord api documentation for requesting specific routes.

Parameters:

  • state table The api state.
  • ... Parameters to the request

Returns:

    api.request

Usage:

    api.get_channel(state, id)
