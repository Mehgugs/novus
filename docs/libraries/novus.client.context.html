Interposition mechanism for adding new methods to the context object.

  • name string The method name to interpose.

    func The old method, if it exists.

objects

context
An object for holding the various objects emitted in events.

Fields:

  • guild The guild involved in the request.
  • channel The channel involved in the request.
  • user The user object involved in the request.
  • member The member object involved in the request.
  • role The role involved in the request.
  • msg The msg involved in the request.

Usage:

    -- Example
client.events.MESSAGE_CREATE :listen (function(ctx)
    ctx.channel:send{embed = {
        title = ctx.guild.name
        description = "%s sent a message." % ctx.user
    }}
end)
