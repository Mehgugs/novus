Functions

to_permission (num_or_str)
Converts a string permission name to a permission integer, does nothing to permission integers.

Parameters:

  • num_or_str integer or string Input value.

Returns:

    integer The permission integer

Or

    nil Nil if there is no corresponding integer.

Usage:

    permission.to_permission("administrator") --> 0x8
resolve (p)
Resolves a permission placeholer into its value.

Parameters:

  • p placeholder or integer The permission value.

Returns:

    integer The numerical permission value.
ctor (...)
Given a vararg list of permission names / integers, returns the permission integer for each. Calling the module table will call this via a metamethod. Unrecognized values are coerced to permission.NONE.

Parameters:

  • ... integer or string Permission integers or permission names.

Returns:

    integer The permission integers.

Usage:

    permission.ctor('sendMessages', 'administrator') --> 0x800, 0x008
permission"sendMessages" --> 0x800

Returns:

    number The permission values.

Usage:

    permission.construct('sendMessages', 'administrator') --> 0x808

complement (a, ...)
Given a permission value a and a vararg list of permission values, returns a without the permission values in the vararg.

Returns:

    number The permission value.

    permission.complement(0x808, 0x8) --> 0x800
enable (p, ...)
Given a permission value p and a vararg list of permissions, sets the permissions (BORs them with p).

Parameters:

  • p integer The permission value.
  • ... integer Permissions.

Returns:

    number The permission value.

Usage:

    permission.enable(0x800, 'viewAuditLog', 'administrator') --> 0x888

Returns:

    number The permission value.

Usage:

    permission.disable(0x888, 'viewAuditLog', 'administrator') --> 0x800

Objects

placeholder
A permission placeholder value. This is a stateful container for a permission value. You can use a placeholder as a permission value in any of this module’s functions.

Fields:

  • value integer The current permission value.

See also:

    permission.new

Usage:

    placeholder = permission.new()
placeholder:enable('administrator', 'sendMessages', 'viewAuditlog')
placeholder:decompose() --> {'administrator', 'sendMessages', 'viewAuditlog'}
placeholder.value --> 0x888
