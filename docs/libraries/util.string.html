Module util.string

String utilities.

Dependencies: novus.util.table

Functions

startswith (self, s) Check if a string starts with the given string s.
endswith (self, s) Check if a string ends with the given string s.
suffix (self, pre) Returns the rest of the subject string after the starting value pre.
prefix (self, pre) Returns the start of the subject before the ending value pre.
levenshtein (str1, str2) Computes the Levenshtein distance between two strings.
inject () Adds the module functions into the global string table, this is called automatically by util
__mod (value) Shorthand for formatting a string.

__mod (value)
Shorthand for formatting a string. If value is a table it is unpacked into the format.

Parameters:

  • value A value to format into str.

Usage:

    "%d" % 2 --> "2"
"%d, %d, %d" % {1, 2, 3} --> "1, 2, 3"
