Module util.lpeg

Lpeg combinators and utilities.

Functions

exactly (n, p) Returns an Lpeg pattern which matches exactly n times.
just (p) Returns an Lpeg pattern which matches just the input pattern or nothing.
check (p) Returns an Lpeg pattern which captures true when it matches or false otherwise.
sepby (s, p) Returns a pattern which matches one or more of: the input pattern separated by the pattern s.
endby (s, p) Returns a pattern which matches one or more of: the input pattern followed by the pattern s.
between (b, s) Returns a pattern which matches zero or more of the input pattern, between the pattern s.
anywhere (p) Returns a pattern which matches the input pattern anywhere in the string.
