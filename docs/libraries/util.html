Module util

Utilities.

Dependencies: util.string, util.table.

This module has an __index metamethod which will load other utilities: util.lpeg loads the module novus.util.lpeg.

shift (n, start1, stop1, start2, stop2)
Takes a value n in the range start1, stop1 (inclusive) and converts it into a value in the range start2, stop2.

Parameters:

  • n number The input value.
  • start1 number The start of the range the value n is in.
  • stop1 number The end of the range.
  • start2 number The start of the new range.
  • stop2 number The end of the new range.

Returns:

    number The new value in the range [start2, stop2]
