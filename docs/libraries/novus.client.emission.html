Module novus.client.emission

Event emission.

Dependencies: novus.util

Functions

new () Makes a new emitter.
listen (em, ...) Adds one or more listeners to the emitter.
emit (em, event) Emits an event calling all the callbacks registered.

remove (em, cb) Removes a callback by value, filtering the queue.

__pairs (em[, timeout])
pairs iterator, iterates over all events received until timeout or event data is nil. You may set the timeout to 0 to effectively stop receiving events.

Parameters:

  • em emitter
  • timeout number (optional)

Usage:

    for ctx, event in pairs(emitter) do
   -- process event here
   if condition then ctx.timeout = 0 end --unsubscribe
end

objects

emitter
An emitter object. Set callbacks with listen and emit events with emit. All functions which take a emitter as their first argument can be called from the emitter in method form.

Fields:

  • pollfd Condition variable, signalled on emission.

Usage:

    E = emission.new()
E:listen(print)
E:emit"Hello, world!"
--> Hello, world!
E:remove(print)
E:emit"No one's listening :("
--> nothing happens
